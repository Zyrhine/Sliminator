## Getting Started
### Prerequisites
* Unity 2020.3.14f1 (LTS)

### Project Installation
1. Install **Unity Hub**
2. Add Unity Version 2020.3.14f1 (LTS)
3. Clone the repo
   ```sh
   git clone https://github.com/Zyrhine/Sliminator.git
   ```
4. Add the repo folder in Unity Hub (Projects -> Add)
5. Launch!
